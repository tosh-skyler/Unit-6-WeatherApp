# Weather Application

## Function 

* Get the user location on load
* Get the user via user input
* Get the current City & State
* Get the current temperature
* Display the following; icon image, humidity, wind speed, UV index.
* Get five day forecast displaying the following; date, icon image, temperature, humidity.
* Save search history to local storage.

## Technologies Used

* Google Geolocation API
